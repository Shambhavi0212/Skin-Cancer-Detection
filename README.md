
# Skin Cancer Detection

## Project Overview
This project uses machine learning to detect and classify skin cancer from images of skin lesions. The goal is to assist medical professionals in early diagnosis by analyzing images to determine if a lesion is benign or malignant.

## Features
- **Image-based analysis**: The application analyzes images of skin lesions to predict if they are cancerous.
- **Machine learning model**: A trained machine learning model (`cancer_detection_model.h5`) is used to classify the lesions.
- **Data visualization**: Visual feedback on the input image and model prediction results.

## Installation
To get started with the project, clone this repository and install the required dependencies.

1. Clone the repository:
   - `git clone https://github.com/Shambhavi0212/Skin-Cancer-Detection.git`

2. Install the necessary dependencies by using a Python package manager:
   - `pip install -r requirements.txt`

## Usage
1. Run the application by executing the main script.
2. Upload or provide an image of a skin lesion to be analyzed by the model.
3. The model will predict whether the lesion is benign or malignant and display the result.

## Dependencies
- Python 3.12.0
- TensorFlow
- Keras
- OpenCV
- Matplotlib
- Other libraries included in `requirements.txt`
